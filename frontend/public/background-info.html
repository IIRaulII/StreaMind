Fondo de Pósters de Películas

Este archivo proporciona información sobre la imagen de fondo utilizada en la aplicación StreaMind.

Uso en la aplicación

La imagen se utiliza como fondo en toda la aplicación, con un overlay oscuro para mejorar la legibilidad del contenido. Se implementa mediante CSS en el archivo style.css.

Técnica de implementación

Para mostrar la imagen como fondo con un overlay oscuro, se usa la siguiente técnica CSS:

body {
  background-image: url('/movie-posters-background.jpg');
  background-size: cover;
  background-position: center;
  background-attachment: fixed;
  background-repeat: no-repeat;
}

body::before {
  content: '';
  position: fixed;
  top: 0;
  left: 0;
  width: 100%;
  height: 100%;
  background-color: rgba(18, 18, 18, 0.92);
  z-index: -1;
}

Esta implementación asegura que la imagen de fondo se muestre correctamente en todos los dispositivos mientras mantiene la legibilidad del contenido.
